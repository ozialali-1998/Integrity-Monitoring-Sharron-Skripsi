# FIM Frontend

React + Vite + Tailwind frontend for the File Integrity Monitoring research dashboard.

## Current behavior

The frontend uses `src/services/mockApi.js` and does not call the backend yet. It includes the requested pages:

- Dashboard
- Directory Monitoring
- Verification
- Benchmark
- Logs
- Settings

## Run

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
npm run build
```
